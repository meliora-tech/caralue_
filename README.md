## Getting Started

 - Have a MongoDB URI.

 - Run the scrapers in the scrapers folder. Create also a data folder to save all scraped info.

 - Upload the data into your mongo collection. Run `mongo_cars.py`

 - Start the application: <code> python app.py </code>
